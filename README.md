# sbpy.org
sbpy website


# How to

## Initial setup
Requires: Node.js

  1. `npm install -g firebase-tools` 
  
  1. `firebase login`
  
  1. Clone sbpy.org repo.
  
  1. cd sbpy.org
  
  1. `firebase init` and select sbpy-website project.

## Update

  1. Make changes
  
  1. Test serving locally `firebase serve`
  
  1. Make changes...test...changes...test
  
  1. Commit to repo
  
  1. Deploy `firebase deploy`
